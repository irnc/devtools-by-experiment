# Navigation

- `Ctrl + [` switches to the left tab
- `Ctrl + ]` switches to the right tab

# View

- `Ctrl + =` and `Ctrl + +` increases font size of DevTools UI
- `Ctrl + -` and `Ctrl + _` decreases font size of DevTools UI
- `Ctrl + 0` resets to normal zoom level
